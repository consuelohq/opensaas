## implementation notes

### approach

**backend:** add GET /v1/voice/status to voice.ts (before param routes). uses getWorkspaceTwilioConfig + 
getDecryptedCredentials + isHostedInstance from twilio-config.ts to determine status without throwing.
tries to build dialer and list numbers to check twilioConnected + hasPhoneNumbers.

**frontend:**
- TwilioConfigStatus type in dialer/types/dialer.ts
- twilioConfigStatusState atom in dialer/states/
- useTwilioConfigStatus hook — fetches /v1/voice/status, polls every 5s when unconfigured
- TwilioConfigStatus component — renders appropriate message per mode/state
- DialerSidebar — renders TwilioConfigStatus, hides DialPad/CallButton when unconfigured
- useTwilioDevice — reads twilioConfigStatusState, skips initDevice when not configured

## implementation notes

## research findings

- site uses Tailwind v4 with CSS vars: --color-bg, --color-fg, --color-surface-0, --color-surface-1, --color-muted, --color-border
- dark mode via prefers-color-scheme media query in global.css
- all components use framer-motion useInView for scroll-triggered animations
- lucide-react for icons, named exports only
- section pattern: py-20 sm:py-24, mx-auto max-w-[1000px] px-6
- framer-motion already installed — no new deps needed

## approach

- pure SVG + framer-motion for workflow viz (no reactflow — overkill for landing page)
- workflow steps: New Lead → Auto-Dial → AI Coaching → Log to CRM → Follow Up
- nodes: rounded cards with icon + label + type badge (trigger/action)
- connectors: simple div-based lines with animated dots via framer-motion
- desktop: horizontal flow, mobile: vertical stack
- all colors via CSS variables for automatic dark/light mode support

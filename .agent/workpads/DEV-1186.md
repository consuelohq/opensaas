### research: CRM workflow node anatomy
- WorkflowNodeContainer: rounded-md border, 240px max, padding 8px, cursor pointer
- WorkflowNodeIconContainer: 32x32 rounded-sm, transparent light bg, centered icon
- WorkflowNodeLabel: type badge (Trigger/Action) with color coding
- WorkflowNodeTitle: font-size md, font-weight medium, single line ellipsis
- edges: BaseEdge with stroke color based on selection state
- colors: blue (default), green (success), yellow (running), red (failed)
- canvas: ReactFlow with Background dots, reset default styles

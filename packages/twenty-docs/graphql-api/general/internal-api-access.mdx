## Overview

Consuelo provides a dedicated GraphQL API for internal team operations. This guide covers how to access and use the internal Consuelo instance.

## Internal Instance

**Workspace ID:** `7d0894c1-bdb1-4dd6-9a00-78681b52d5f6`

### Endpoints

| Endpoint | URL |
|----------|-----|
| **GraphQL** | `https://consuelo.consuelohq.com/graphql` |
| **Metadata** | `https://consuelo.consuelohq.com/metadata` |
| **App** | `https://consuelo.consuelohq.com` |

### Authentication

Use your internal API key:

```
Authorization: Bearer YOUR_INTERNAL_API_KEY
```

Get your key from **Settings → APIs & Webhooks** in the internal workspace.

## Testing the API

### Using cURL

```bash
curl -X POST https://consuelo.consuelohq.com/graphql \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "{ currentUser { id email } }"}'
```

### Using GraphQL Playground

1. Open `https://consuelo.consuelohq.com/graphql`
2. Click **HTTP HEADERS** at the bottom
3. Add your authorization header:
   ```json
   {
     "Authorization": "Bearer YOUR_API_KEY"
   }
   ```
4. Run queries in the editor

### Using Postman

1. Set request type: **POST**
2. URL: `https://consuelo.consuelohq.com/graphql`
3. Headers:
   - `Authorization: Bearer YOUR_API_KEY`
   - `Content-Type: application/json`
4. Body (raw JSON):
   ```json
   {
     "query": "{ people(first: 10) { edges { node { id email } } } }"
   }
   ```

## Common Query Patterns

### Query People

```graphql
query GetPeople($first: Int!) {
  people(first: $first) {
    edges {
      node {
        id
        email
        firstName
        lastName
        company {
          id
          name
        }
      }
    }
  }
}
```

**Variables:**
```json
{
  "first": 10
}
```

### Query Companies

```graphql
query GetCompanies($filter: CompanyFilterInput) {
  companies(filter: $filter, first: 100) {
    edges {
      node {
        id
        name
        domain
        industry
        employees
        createdAt
      }
    }
    totalCount
  }
}
```

**Variables:**
```json
{
  "filter": {
    "industry": {
      "eq": "Technology"
    }
  }
}
```

### Create Record

```graphql
mutation CreatePerson($data: PersonCreateInput!) {
  createPerson(data: $data) {
    id
    email
    firstName
    lastName
  }
}
```

**Variables:**
```json
{
  "data": {
    "email": "new@example.com",
    "firstName": "New",
    "lastName": "Contact"
  }
}
```

### Update Record

```graphql
mutation UpdatePerson($id: ID!, $data: PersonUpdateInput!) {
  updatePerson(id: $id, data: $data) {
    id
    email
    firstName
    lastName
    updatedAt
  }
}
```

**Variables:**
```json
{
  "id": "person-uuid-here",
  "data": {
    "firstName": "Updated Name"
  }
}
```

### Delete Record

```graphql
mutation DeletePerson($id: ID!) {
  deletePerson(id: $id) {
    id
  }
}
```

## Rate Limits

| Limit | Value |
|-------|-------|
| **Requests** | 100 per minute |
| **Batch size** | 60 records per call |

<Warning>
Hitting rate limits returns HTTP 429. Implement retry logic with exponential backoff.
</Warning>

## Error Handling

### Common Errors

| Code | Meaning | Solution |
|------|---------|----------|
| `401` | Unauthorized | Check API key |
| `403` | Forbidden | Check permissions |
| `404` | Not found | Verify record ID |
| `429` | Rate limited | Slow down, retry |
| `500` | Server error | Retry or report |

### Response Format

**Success:**
```json
{
  "data": {
    "people": {
      "edges": [...]
    }
  }
}
```

**Error:**
```json
{
  "errors": [
    {
      "message": "Unauthorized",
      "extensions": {
        "code": "UNAUTHORIZED"
      }
    }
  ]
}
```

## SDK Usage

### Node.js Example

```javascript
import { GraphQLClient, gql } from 'graphql-request';

const client = new GraphQLClient('https://consuelo.consuelohq.com/graphql', {
  headers: {
    authorization: `Bearer ${process.env.CONSUELO_API_KEY}`,
  },
});

const query = gql`
  query GetPeople {
    people(first: 10) {
      edges {
        node {
          id
          email
        }
      }
    }
  }
`;

const data = await client.request(query);
console.log(data);
```

### Python Example

```python
import requests

url = "https://consuelo.consuelohq.com/graphql"
headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

query = """
query GetPeople {
  people(first: 10) {
    edges {
      node {
        id
        email
      }
    }
  }
}
"""

response = requests.post(
    url,
    headers=headers,
    json={"query": query}
)

data = response.json()
print(data)
```

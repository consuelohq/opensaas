---
title: Install in HighLevel
description: Install the Consuelo embedded dialer for a HighLevel location.
---

Use this guide when you are installing the embedded dialer for the first time.

## Step 1: install the marketplace app

A HighLevel admin should install the Consuelo app for the target location from the HighLevel marketplace.

## Step 2: confirm the embedded URL

The embedded widget is served from:

https://calls.consuelohq.com/calls/embedded

That is the URL the HighLevel sidebar should open for the dialer experience.

## Step 3: open the app from the HighLevel sidebar

After installation, open the dialer from the HighLevel sidebar and confirm the widget loads in-place.

## Step 4: grant browser permissions

Allow microphone access in the browser and verify that your operating system also allows the browser to use your microphone.

## Step 5: validate a real workflow

Run a short validation pass:

- open a known contact in HighLevel
- launch the embedded dialer
- start a call from the contact context
- finish the call and confirm it appears on the contact timeline

## If the app does not appear

Go to [Troubleshoot the Embedded Dialer](/user-guide/highlevel/embedded/how-tos/troubleshoot-embed) and start with the sidebar visibility checks.

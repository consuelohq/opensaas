> Make calls directly from the CRM with the built-in dialer.

The Consuelo dialer is a sidebar that provides full telephony functionality directly within the CRM. Access it anytime using keyboard shortcuts or the navigation bar.

## Opening the Dialer

**Keyboard Shortcut:**

- Mac: `Cmd+D`
- Windows: `Ctrl+D`

**Alternative:**

- Click the phone icon in the top navigation bar

## Dialer Layout

The dialer sidebar consists of several key areas:

### 1. Dial Pad

The main area for entering phone numbers:

- Numeric keypad (0-9)
- Asterisk (\*) and pound (#) keys
- Clear button
- Call button

### 2. Caller ID Selector

Dropdown to select which phone number displays when calling:

- Shows all configured caller IDs
- Indicates local presence status
- Quick access to add new numbers

### 3. Contact Search

Search for contacts while the dialer is open:

- Type to search by name or phone number
- Shows matching results in real-time
- Click a result to auto-fill the number

### 4. Active Call Controls

When on a call, these controls appear:

- **Mute** - Toggle microphone on/off
- **Hold** - Place caller on hold
- **Transfer** - Initiate a call transfer
- **Keypad** - Access DTMF tones
- **Hang Up** - End the call

### 5. Call History

Recent calls list showing:

- Call direction (inbound/outbound)
- Contact name and number
- Duration and timestamp
- Call status (completed, missed, no answer)

## Keyboard Shortcuts

| Shortcut           | Action                       |
| ------------------ | ---------------------------- |
| `Cmd+D` / `Ctrl+D` | Open/close dialer            |
| `Enter`            | Initiate call                |
| `M`                | Toggle mute                  |
| `H`                | Toggle hold                  |
| `T`                | Open transfer                |
| `Escape`           | Close dialer / Cancel action |

<Note>
  Both methods use the same Consuelo infrastructure. Calls made via Discord
  appear in the CRM call history, and vice versa.
</Note>

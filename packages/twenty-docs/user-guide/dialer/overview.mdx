## Which to Use?

**Use the CRM Dialer when:**

- You need the full CRM context visible
- You prefer a visual interface
- You're working at your desk

**Use the Discord Bot when:**

- Your team communicates in Discord
- You want team visibility on calls
- You prefer chat-based workflows
- You need real-time coaching via whisper

<Note>
  Both methods use the same Consuelo infrastructure. Calls made via Discord
  appear in the CRM call history, and vice versa.
</Note>

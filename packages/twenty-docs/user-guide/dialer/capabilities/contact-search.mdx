> Search contacts and review call history without opening the CRM.

The Discord bot provides quick access to your CRM data, letting you look up contacts and review recent calls directly from chat.

## Contact Search

Search your workspace contacts by name, company, or phone number.

```
/consuelo contacts search <query>
```

**Examples:**

```
/consuelo contacts search john
/consuelo contacts search acme corp
/consuelo contacts search 555-1234
```

**Results:**

```
🔍 Contacts: "john"

👤 John Smith — (555) 123-4567 • Acme Corp • Last call: 2024-01-15
👤 Johnny Lee — (555) 987-6543 • TechStart Inc
👤 John Doe — (555) 456-7890 • Last call: 2024-01-10
```

**What you can see:**

| Field         | Description                 |
| ------------- | --------------------------- |
| **Name**      | Contact's full name         |
| **Phone**     | Primary phone number        |
| **Company**   | Associated company (if any) |
| **Last Call** | Date of most recent call    |

**Limitations:**

- Returns up to 5 results
- Searches name, company, and phone fields
- Only shows contacts you have access to based on workspace permissions

## Call History

View your recent calls.

```
/consuelo history
```

**Output:**

```
📞 Recent Calls

✅ John Smith — 3m 42s — answered — 2024-01-15 14:30
❌ Acme Corp — 0m 0s — no-answer — 2024-01-15 14:25
✅ Jane Doe — 5m 12s — answered — 2024-01-15 14:18
❌ Busy Signal LLC — 0m 0s — busy — 2024-01-15 14:10
✅ Mike Johnson — 2m 30s — answered — 2024-01-15 14:05
```

**Icons:**

| Icon | Meaning                      |
| ---- | ---------------------------- |
| ✅   | Answered call                |
| ❌   | No answer / Busy / Voicemail |

**Details shown:**

- Contact name or phone number
- Call duration
- Call outcome
- Timestamp (YYYY-MM-DD HH:MM format)

## Your Stats

View your personal calling statistics.

```
/consuelo me
```

**Output:**

```
👤 John Smith (You)

Calls Today: 23
Calls This Week: 147
Connect Rate: 68%
Avg Duration: 4m 12s
```

**Metrics:**

| Metric              | Description                            |
| ------------------- | -------------------------------------- |
| **Calls Today**     | Total calls made today                 |
| **Calls This Week** | Total calls made this week             |
| **Connect Rate**    | Percentage of calls that were answered |
| **Avg Duration**    | Average length of answered calls       |

<Note>
  Stats are calculated based on calls made through Consuelo (both via Discord
  bot and CRM dialer).
</Note>

Consuelo offers multiple ways to initiate calls depending on your workflow.

## Methods for Making Calls

### 1. Dial Pad

Use the on-screen keypad to enter a phone number:

1. Open the dialer (`Cmd+D` / `Ctrl+D`)
2. Enter the number using the keypad or keyboard
3. Select your caller ID
4. Click **Call** or press `Enter`

### 2. Click-to-Call

Call contacts directly from their record:

1. Navigate to any contact
2. Click the phone icon next to the contact's number
3. The dialer opens with the number pre-filled
4. Click **Call** to initiate

### 3. Quick Search

Search and call from anywhere:

1. Press `Cmd+K` / `Ctrl+K` to open quick search
2. Type a contact name or phone number
3. Select the contact from results
4. Click the phone icon to call

## Caller ID Selection

When making calls, you can choose which phone number displays to the recipient:

### Manual Selection

1. Open the dialer
2. Click the caller ID dropdown
3. Select your desired number

### Local Presence (Automatic)

Enable local presence to automatically display a number with the same area code as the recipient:

1. Go to **Settings** → **Dialer Settings**
2. Enable **Local Presence**
3. Consuelo will:
   - Detect the area code from the number being called
   - Select a caller ID with a matching or nearby area code
   - Improve answer rates by showing local numbers

<Note>
  Local presence requires multiple caller IDs configured with different area
  codes.
</Note>

## Call Queue

For outbound calling campaigns:

1. Add contacts to a queue in **Settings** → **Call Queue**
2. Enable parallel dialing (up to 3 concurrent calls)
3. Consuelo will automatically dial the next contact when a line becomes available

## Best Practices

- Use local presence for cold calls to improve answer rates
- Keep caller IDs consistent for brand recognition
- Review call history to avoid calling uninterested prospects multiple times

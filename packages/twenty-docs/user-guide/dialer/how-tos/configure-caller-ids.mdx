This guide walks you through adding phone numbers to your workspace, setting a primary caller ID, and provisioning numbers for local presence coverage.

## Add a phone number

<Steps>
  <Step title="Open phone number settings">
    Go to **Settings → Dialer → Phone Numbers**.
  </Step>
  <Step title="Click Add Number">
    The search interface opens.
  </Step>
  <Step title="Search by area code">
    Enter the area code you want (e.g. `713` for Houston). The system calls the Twilio API to find available numbers in that area code.
  </Step>
  <Step title="Select a number">
    Choose from the available results. Each shows the full phone number.
  </Step>
  <Step title="Provision">
    Click **Provision**. The number is purchased from Twilio and added to your workspace. It appears in the caller ID dropdown immediately.
  </Step>
</Steps>

<Note>
Twilio charges a monthly fee per provisioned number (typically $1-2/month for US local numbers). Check [Twilio's pricing page](https://www.twilio.com/en-us/voice/pricing) for current rates.
</Note>

## Set a primary number

Your primary number is the default caller ID used when no other selection is made.

1. Go to **Settings → Dialer → Phone Numbers**
2. Find the number you want as primary
3. Click **Set as Primary**

The primary number is pre-selected in the caller ID dropdown when you open the dialer.

## Provision numbers for local presence

Local presence works by matching the recipient's area code to one of your provisioned numbers. The more area codes you cover, the better your match rate.

### Strategy: cover your target markets

If your team calls into specific regions, provision numbers for the top area codes in those regions:

| Region | Area codes to cover |
|---|---|
| Texas | `214` (Dallas), `512` (Austin), `713` (Houston), `817` (Fort Worth), `210` (San Antonio) |
| California | `213` (LA), `415` (SF), `619` (San Diego), `408` (San Jose), `916` (Sacramento) |
| New York | `212` (Manhattan), `718` (Brooklyn), `516` (Long Island), `914` (Westchester) |
| Florida | `305` (Miami), `407` (Orlando), `813` (Tampa), `904` (Jacksonville) |

### How many numbers do you need?

- **Minimum**: 1 number (no local presence, but calling works)
- **Basic local presence**: 5-10 numbers covering your top markets
- **Strong local presence**: 15-20 numbers covering all major metros you call into
- **Don't over-provision**: each number costs money monthly. Cover your top area codes, not every code in the country.

### Preview your coverage

Use the preview tool in settings to test your local presence matching:

1. Go to **Settings → Dialer**
2. Scroll to the **Local Presence** section
3. Enter a phone number in the **Preview Caller ID** field
4. Click **Preview**
5. The system shows which of your numbers would be selected and whether it's an exact area code match

## Release a number

To stop paying for a number you no longer need:

1. Go to **Settings → Dialer → Phone Numbers**
2. Find the number
3. Click **Release**
4. Confirm the action

<Warning>
Releasing a number is permanent. The number goes back to Twilio's pool and may be assigned to someone else. You cannot get it back. If this is your only number, you won't be able to make calls until you provision a new one.
</Warning>

> Use the Consuelo dialer, contacts, and AI coaching directly in your browser with the Chrome Extension.

The Consuelo Chrome Extension brings your sales tools into any browser tab. Make calls, manage contacts, run queues, and get real-time AI coaching without switching windows.

## Key Features

| Feature                  | Description                                                        |
| ------------------------ | ------------------------------------------------------------------ |
| **Side Panel Dialer**    | Full dialer in Chrome's side panel — stays open across tabs        |
| **Popup Mode**           | Quick-access popup for fast calls and contact lookup               |
| **Google Sheets Import** | Import contacts directly from Google Sheets into your CRM          |
| **Call History**         | View recent calls with duration, outcome, and coaching notes       |
| **Real-Time Coaching**   | AI-powered tips during active calls based on conversation context  |
| **Queue Management**     | Create and run dialing queues with progress tracking               |
| **Caller ID Selection**  | Choose which number to call from before dialing                    |

## Quick Start

1. **[Install the extension](/user-guide/chrome-extension/getting-started#installation)** from the Chrome Web Store
2. **[Sign in](/user-guide/chrome-extension/getting-started#sign-in)** with your Consuelo account
3. **Make your first call** using the side panel dialer

<Note>
  The Chrome Extension requires a Consuelo account with an active subscription and at least one provisioned phone number.
</Note>

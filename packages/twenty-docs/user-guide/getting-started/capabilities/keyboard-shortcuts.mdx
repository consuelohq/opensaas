title: Keyboard Shortcuts
description: Navigate Consuelo faster with keyboard shortcuts. Every shortcut is designed to keep your hands on the keyboard and your workflow uninterrupted.
---

Consuelo is built for keyboard-driven workflows. Press `⌘ /` (Mac) or `Ctrl /` (Windows) anywhere in the app to open the shortcuts panel.

## General

These shortcuts work everywhere in Consuelo.

| Shortcut | Action |
|----------|--------|
| `⌘ K` | Open command menu |
| `/` | Search records |
| `@` | Ask AI |
| `⌘ /` | Open keyboard shortcuts panel |
| `Esc` | Back / Close |
| `⌘ ,` | Open settings |
| `⇧ F` | Mark as favourite |
| `?` | Open keyboard shortcut help |

## Navigation

Use `G` followed by a letter to jump to any section. These work from any page.

| Shortcut | Action |
|----------|--------|
| `G` then `P` | Go to People |
| `G` then `C` | Go to Companies |
| `G` then `D` | Go to Dashboards |
| `G` then `T` | Go to Tasks |
| `G` then `N` | Go to Notes |
| `G` then `W` | Go to Workflows |
| `G` then `S` | Go to Settings |
| `G` then `A` | Go to Computer |

:::tip
Navigation shortcuts are also shown on sidebar items when you hover over them.
:::

## Record Table

These shortcuts are active when you're viewing a table or list of records.

| Shortcut | Action |
|----------|--------|
| `→` | Move right |
| `←` | Move left |
| `↑` | Move up |
| `↓` | Move down |
| `Enter` | Open focused record |
| `Esc` | Clear selection |
| `X` | Select record |
| `⇧ Click` | Select multiple records |

## Record Actions

These shortcuts are active when a record is focused or selected.

| Shortcut | Action |
|----------|--------|
| `Space` | Peek into record |
| `Enter` | Open record |
| `⌘ ⌫` | Delete record |
| `⌘ ⇧ ,` | Copy record URL |

## Dialer

These shortcuts are active when the dialer sidebar is open.

| Shortcut | Action |
|----------|--------|
| `⌘ D` | Toggle dialer sidebar |
| `M` | Mute / Unmute |
| `H` | Hold / Unhold |
| `T` | Transfer |
| `Esc` | End call (during call) / Close dialer (when idle) |
| `⌘ ⇧ C` | Call selected contact |
| `⌘ ⇧ M` | Toggle calling mode |

:::info
On Windows and Linux, replace `⌘` with `Ctrl` for all shortcuts above.
:::

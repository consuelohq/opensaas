---
title: Set Up Pipeline Sync
description: Map GHL stages to Consuelo stages and run the opportunity sync.
---

Use this guide after the GHL connection is active and you are ready to sync stage movement.

## Step 1: connect the GHL location

Pipeline sync depends on the same OAuth connection used for contacts. Do not start stage mapping until the integration shows a connected location.

## Step 2: load your GHL pipelines

Open the pipeline mapping section and wait for Consuelo to load the pipelines and stages returned by GoHighLevel.

## Step 3: map each GHL stage to a CRM stage

Work stage by stage. Only mapped stages participate in the sync.

## Step 4: save the mappings

After you save, Consuelo stores the stage-level mapping for the workspace.

## Step 5: run the pipeline sync

Trigger the sync once the mappings are in place. The sync job:

- fetches GHL opportunities
- skips unmapped stages
- creates or updates local opportunity sync records

## Step 6: review conflicts if needed

If you want to inspect differences before a larger run, use the conflict check option. That compares the stored GHL stage against the current CRM stage for synced opportunities.

## Common setup mistakes

- mapping only the pipeline and forgetting the stages
- leaving active GHL stages unmapped
- expecting opportunities in unmapped stages to sync automatically

title: Import Contacts from GHL
description: Run an initial GoHighLevel contact import and review the sync log.
---

Use this guide for the first import from GoHighLevel into Consuelo.

## Step 1: confirm the connection is healthy

Before importing, make sure the integration shows a connected state for the correct GHL location.

## Step 2: add a tag filter if you want a smaller first run

The settings screen accepts a comma-separated tag filter. If you provide tags, only contacts carrying at least one of those GHL tags are included in the import.

## Step 3: click Import All Contacts

The importer paginates through the GHL contacts list in batches of 100 until there are no more contacts to fetch.

## Step 4: watch the import progress

The settings page shows import progress, then writes the final result to the sync log.

Look for these numbers when the import finishes:

- imported
- updated
- skipped
- errors

## Step 5: spot-check imported records

Review a small batch of imported People records and verify:

- phone numbers are normalized correctly
- names and emails landed where expected
- existing records were updated instead of duplicated when they matched by phone or email

## Step 6: review the sync log

Use the sync log to confirm whether the import needs cleanup before you run the next sync.

## If too many records were skipped

Check these things first:

- whether the tag filter was too narrow
- whether those contacts already had sync mappings
- whether the contacts matched existing People records by normalized phone or email

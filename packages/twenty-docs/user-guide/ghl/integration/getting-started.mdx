Use this guide to connect a GoHighLevel location to Consuelo for the first time.

## Before you connect

Make sure you have:

- access to the Consuelo workspace where the integration should live
- a GoHighLevel user who can authorize access for the target location
- a plan for your first import so you know whether to bring everything in or start with a filtered subset

## Step 1: open the GHL integration settings

In Consuelo, open the GoHighLevel integration settings page and click **Connect GHL**.

## Step 2: approve the OAuth flow

Consuelo starts the GoHighLevel OAuth flow and sends you to the location chooser. Approve the connection for the correct location, then return to Consuelo.

## Step 3: confirm the connection state

After the callback completes, the integration should show that the location is connected.

A healthy connection gives you:

- a connected state instead of disconnected
- the location identifier returned by GoHighLevel
- a connected timestamp

## Step 4: run your first contact import

Start with a first import so Consuelo can build the sync map between GHL contact ids and your People records.

If you want to limit the first run, use a tag filter before you import.

## Step 5: review the sync log

After the import completes, review the sync log to confirm:

- how many records were imported
- how many were updated instead of created
- how many were skipped
- whether any errors need cleanup before the next run

## What permissions the integration asks for

The OAuth flow requests the permissions required for contacts, conversations, messages, and location access inside GoHighLevel.

> The embedded dialer gives GoHighLevel users a full calling surface without leaving GHL.

The embedded dialer is the in-GHL experience. GoHighLevel loads Consuelo inside an iframe, the widget validates that it is running in an embedded context, and the app keeps contact context from GHL available while you call, queue leads, or review history.

## How it works

1. GoHighLevel opens the widget in an iframe.
2. The embedded entry page marks the app as embedded-only and checks that it is not running as a standalone browser tab.
3. The widget asks the parent frame for contact context and sign-in data.
4. The dialer receives click-to-call, queue-add, or view-history intents from GHL.
5. During and after the call, the widget can send call state and logging events back to the parent frame.

## Deep link actions

The embedded app handles three high-level intents from GoHighLevel:

| Action | What it does |
| --- | --- |
| call | Opens the dialer with the selected contact and can auto-dial when requested |
| queue-add | Creates a queue from one or many contacts passed from GHL |
| view-history | Opens the widget with the selected contact in context for history review |

## Who this is for

Use the embedded dialer if you want reps to stay inside GoHighLevel while they call. If you need contact import, sync logs, or pipeline mapping, continue to the [integration section](/user-guide/ghl/integration/overview).

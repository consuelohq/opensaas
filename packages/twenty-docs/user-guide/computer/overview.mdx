## What is Computer?

Computer is Consuelo's AI assistant built specifically for sales teams. It works alongside you during calls, analyzes conversations, and handles repetitive CRM tasks — so you can focus on selling.

Unlike generic AI chatbots, Computer has deep integration with your CRM data:

- **Knows your contacts** — understands who you're calling, their company, their deal stage
- **Sees your pipeline** — knows which opportunities are closing, which ones are stuck
- **Understands context** — recognizes when a call is happening and what coaching you need
- **Takes action** — logs calls, updates deals, creates tasks, queues follow-ups

## Key Capabilities

### Real-Time Coaching

Get live coaching during calls with actionable phrases, objection handling, and clarifying questions based on what's being said right now.

<Check>Phrase suggestions appear as the customer speaks</Check>
<Check>Objection handling when they push back</Check>
<Check>Pain funnel questions to uncover needs</Check>

### Post-Call Analysis

After every call, Computer automatically generates analytics, logs outcomes, and creates follow-up tasks.

<Check>Call sentiment and key moments</Check>

<Check>Automatic call logging in CRM</Check>
<Check>Deal stage updates with evidence</Check>

### CRM Actions

Computer can query and modify your CRM data directly:

| Action              | What It Does                                 |
| ------------------- | -------------------------------------------- |
| Search Contacts     | Find contacts by name, email, phone, company |
| Get Contact Details | Retrieve full contact record with notes      |
| List Deals          | View pipeline by stage                       |
| Log Call            | Record call outcome and notes                |
| Update Deal         | Change stage, amount, or notes               |
| Create Note         | Add notes to contact records                 |
| Create Task         | Schedule follow-ups                          |
| Add to Queue        | Queue contacts for calling                   |

### Context Awareness

Computer understands where you are in the app and what you're working on:

- **Contact context** — when viewing a contact, Computer knows their history, company, and deals
- **Deal context** — when viewing an opportunity, Computer sees the stage, amount, and activity
- **Pipeline context** — Computer understands your pipeline stages and what needs attention

## How Computer Works

### During Calls

When you're on a call, Computer activates automatically:

1. **Detects the call** — recognizes when you're connected to a contact
2. **Loads context** — pulls contact info, deal history, recent notes
3. **Listens** — transcribes the conversation in real-time
4. **Coaches** — provides actionable suggestions based on what's being said
5. **Suggests tools** — recommends mute, hold, or transfer based on conversation flow

### After Calls

When a call ends, Computer automatically:

1. **Analyzes** — extracts key moments, sentiment, performance metrics
2. **Logs** — creates a call record with outcome and notes
3. **Updates** — moves deals forward if there's explicit evidence
4. **Tasks** — creates follow-up tasks for promised actions

### On Demand

You can also interact with Computer directly in the agent panel:

- Ask questions about your data ("What opportunities are closing this month?")
- Request actions ("Log a call with John Smith")
- Get insights ("Which deals have been in Negotiation for more than 30 days?")

## Accessing Computer

Computer appears in the sidebar when you need it:

1. **During calls** — opens automatically in coaching mode
2. **Agent panel** — click the Computer icon in the navigation bar
3. **Contact pages** — context-aware suggestions based on the current record

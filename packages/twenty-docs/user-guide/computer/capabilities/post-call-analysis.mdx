## Overview

When a call ends, Computer doesn't just stop. It analyzes the conversation, logs the outcome, updates relevant records, and creates follow-up tasks — all automatically.

## What Happens Automatically

### 1. Call Analytics

Computer extracts key metrics from every call:

| Metric            | Description                                         |
| ----------------- | --------------------------------------------------- |
| **Talk Ratio**    | Percentage of time you spoke vs. the customer       |
| **Response Time** | Average time before you responded                   |
| **Sentiment**     | Overall mood: positive, negative, neutral, or mixed |
| **Trend**         | Did sentiment improve, decline, or stay stable?     |

### 2. Key Moments

Important moments are flagged with timestamps:

```
Key Moments:
├─ 0:42 — Objection raised: "We're already using [Competitor]"
├─ 1:15 — Question: "How does pricing work for teams?"
├─ 3:30 — Commitment: "Let's schedule a demo for Friday"
└─ 4:45 — Insight: "They mentioned they're expanding to Europe"
```

### 3. Call Logging

Every call is automatically logged in the CRM:

- **Contact**: Linked to the correct contact record
- **Outcome**: Answered, voicemail, no answer, or busy
- **Duration**: Total call time
- **Notes**: AI-generated summary of key points
- **Next Step**: Suggested follow-up action

### 4. Deal Updates

If there's evidence of progress, Computer updates the deal:

<Check>Stage advancement when there's explicit commitment</Check>
<Check>Amount updates if new pricing is discussed</Check>
<Check>Notes added with key decision points</Check>

<Warning>
  Computer requires explicit evidence for stage changes. "They seemed
  interested" is not enough. Verbal commitments, scheduled demos, and signed
  agreements count.
</Warning>

### 5. Task Creation

When commitments are made, tasks are created:

| Commitment                               | Task Created                      |
| ---------------------------------------- | --------------------------------- |
| "I'll send you that case study"          | Follow-up: Send case study        |
| "Let's schedule a demo"                  | Schedule: Demo meeting            |
| "Talk to my manager and get back to you" | Follow-up: Check in with decision |

## Analytics Deep Dive

### Sentiment Analysis

Computer tracks sentiment throughout the call:

```
Sentiment Timeline:
Start: Neutral (small talk)
2:00 — Positive (discussed needs)
3:30 — Negative (price objection)
5:00 — Positive (resolved objection)
End: Positive (commitment to demo)

Overall: Mixed, trend: Improving
```

### Performance Metrics

Track your team's performance over time:

- **Average talk ratio** — are reps listening enough?
- **Objection handling score** — how well are objections addressed?
- **Response time** — are reps quick to respond?

### Aggregate Insights

See patterns across all calls:

- Most common objections
- Most effective responses
- Topics that correlate with wins
- Questions that stall deals

## Manual Overrides

You can edit or add to the AI-generated record:

1. Go to the contact's timeline
2. Find the call record
3. Click **Edit** to modify notes or outcome
4. Add manual next steps if needed

## What Computer Won't Do

Computer is helpful but not reckless:

| Won't Do                          | Why                                          |
| --------------------------------- | -------------------------------------------- |
| Change stage on "seemed positive" | Requires explicit commitment                 |
| Create duplicate tasks            | If a similar task exists, it updates instead |
| Log sensitive info                | Only logs business-relevant details          |
| Modify records you can't access   | Respects role permissions                    |

## Tips for Best Results

### Speak Clearly on Calls

Clear speech = better transcription = better analysis.

### State Commitments Explicitly

"Great, I'll send that proposal by end of day today" gets logged as a task. Vague agreements don't.

### Summarize at the End

A quick "So to confirm, you'll review the proposal and we'll talk Friday" helps Computer create accurate tasks.

### Review the Call Record

After calls, quickly check the logged notes and correct anything important.

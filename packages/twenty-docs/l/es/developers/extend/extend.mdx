## Resumen

Consuelo está diseñado para ser extensible. Usa nuestras APIs, webhooks y el framework de aplicaciones para integrarte con tus herramientas existentes y crear funcionalidades personalizadas.

## Lo que puedes hacer

* **APIs**: Consulta y modifica tus datos de CRM de forma programática usando REST o GraphQL
* **Webhooks**: Recibe notificaciones en tiempo real cuando ocurran eventos en Consuelo
* **Aplicaciones**: Crea aplicaciones personalizadas que amplíen las capacidades de Consuelo - Próximamente.

# Sites

Sites is the umbrella for Consuelo OS go-to-market workflows.

It covers the human-facing product surfaces that came from the original GTM platform: records, pipelines, dashboards, workflows, campaigns, dialer surfaces, messaging flows, and pages that humans can use directly while agents work in the background.

## Why Sites exists

Some Consuelo OS tools are mostly agent-facing. Sites is different: it is meant to be useful to humans and agents at the same time.

Use Sites when you need a GTM surface that connects to data, workflows, and agent tools.

## Current docs state

The existing Sites/GTM docs are being moved under this umbrella. Some pages are old and will need editing, but they still describe important product surfaces.

## The math

an agent manually dialing makes **40-60 calls per day.**

a power dialer makes **200+.**

> same agent. same hours. 3-4x more conversations.

## How it works

| step | what happens |
|---|---|
| 1 | load a list of contacts |
| 2 | dialer calls the next number automatically |
| 3 | voicemails and no-answers are skipped |
| 4 | agent only connects to live pickups |
| 5 | notes are logged, next call starts |

no manual dialing. no wasted time listening to rings.

## Local presence

people don't answer unknown numbers.

**local presence** shows a caller ID from the prospect's area code. pickup rates jump from ~5% to ~15%.

| caller ID type | avg pickup rate |
|---|---|
| toll-free (800) | 4-6% |
| out-of-state | 5-8% |
| local area code | 12-18% |

that's the difference between 10 conversations and 30 conversations per day.

## What agents actually care about

- **less downtime** — no manual dialing, no dead air
- **more conversations** — 3x live pickups
- **automatic logging** — call notes saved to CRM
- **disposition codes** — tag outcomes in one click

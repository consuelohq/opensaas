## Table of contents

## The typical sales stack

| tool | cost/seat/month |
|---|---|
| CRM (Salesforce, HubSpot) | $150-300 |
| dialer (Salesloft, Orum) | $150-250 |
| coaching (Gong, Chorus) | $100-200 |
| data enrichment | $50-100 |
| **total** | **$450-850** |

for a 20-person team, that's **$108,000-204,000/year.**

> and you don't own any of it.

## What open source changes

| proprietary | open source |
|---|---|
| per-seat licensing | self-hosted, no seat fees |
| vendor controls your data | you own everything |
| locked into their roadmap | customize anything |
| price increases yearly | costs stay flat |
| integration limits | full API access |

## The three tools that matter

**1. CRM** — contact management, pipeline, deal tracking.

**2. dialer** — power dialing, local presence, call recording.

**3. coaching** — real-time AI prompts, call scoring, compliance.

everything else is optional.

> consuelo bundles all three into one open-source platform.

## Who this is for

- **agencies with 10+ agents** — the cost savings are significant
- **teams that want control** — customize workflows, own your data
- **technical founders** — you can read and modify the source code

## Who this is NOT for

- teams that want zero setup — proprietary tools are easier to start
- solo agents — the cost savings don't justify self-hosting
- agencies that need 500+ integrations out of the box

## Getting started

three paths:

| path | effort | cost |
|---|---|---|
| **cloud hosted** | 5 minutes | free tier available |
| **self-hosted** | 1-2 hours | infrastructure only |
| **custom fork** | days-weeks | your team's time |

most teams start with cloud hosted and move to self-hosted as they scale.

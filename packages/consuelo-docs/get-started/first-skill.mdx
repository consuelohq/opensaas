# Run your first skill

A skill is an approved workflow an agent can run through Consuelo OS.

After local setup, ask your connected agent to check the OS portal and run the first available skill.

```text
Check Consuelo OS and run the daily revenue brief skill.
```

The skill should return a structured result with status, trace information, and any artifact records it creates.

## What should happen

1. The agent asks Consuelo OS for workspace context.
2. Consuelo OS returns available skills and capability health.
3. The agent runs the selected skill through the portal.
4. Consuelo OS records the execution.
5. The result comes back with a trace ID.

# Get Started

Consuelo gives teams two starting paths.

Use **Consuelo OS** when you want to connect agents to your company workspace. OS gives agents a safe portal for context, skills, scripts, files, artifacts, approvals, and memory.

Use **Dialer** when you want to set up calling, queues, caller ID, coaching, and sales workflows inside the Consuelo app.

## Start with OS

Install Consuelo OS locally, connect your agents, run your first skill, and create your first artifact.

## Start with Dialer

Set up your workspace, import contacts, configure caller ID, and make your first call.

## Local and cloud

Local OS runs on your machine. Cloud OS is for shared team workspaces with hosted storage, shared access, and hosted observability.

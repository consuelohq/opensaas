# Local OS and Cloud OS

Consuelo OS can run locally or in the cloud.

## Local OS

Local OS runs on your machine. It is useful for solo operators, internal builders, and people who want their agents connected to local tools and files.

Local OS uses Bun, SQLite, local storage, and a Consuelo OS home folder.

## Cloud OS

Cloud OS is for teams. It uses hosted storage, app-native artifact records, shared access, hosted observability, and managed workspace connections.

## Same model

Local and cloud OS use the same product model: a safe portal, approved skills, executable scripts, tracked artifacts, approval guardrails, and observable executions.

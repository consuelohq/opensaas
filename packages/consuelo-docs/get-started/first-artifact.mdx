# Create your first artifact

Artifacts are app-native outputs created by Consuelo OS.

An artifact can be a report, brief, document, HTML page, PDF, spreadsheet, image, JSON file, or another durable output. The important part is not the file format. The important part is that Consuelo OS tracks who created it, which skill created it, which records were used, and where the output is stored.

## Local artifacts

Local OS writes artifact files under the local OS home and records metadata in SQLite.

## Cloud artifacts

Cloud OS stores artifact bytes in cloud storage and keeps artifact metadata in the app so teams can find, audit, and share outputs.

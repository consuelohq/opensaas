# media.motion.track

Track generic motion vectors and visual features across a video range.

## When to use it

Use this when a claim depends on visible movement, trajectory, or change over time.

## Mental model

Motion tracking measures image movement. It supports trails, comparisons, and visual evidence for breakdowns.

## Inputs

| Field | Meaning |
| --- | --- |
| `input` | Source video path or frame sequence. |
| `range` | Time range to analyze. |
| `out` | Optional output artifact path. |

## Output

Returns motion vectors, tracked points, source references, and analysis metadata.

## Good agent instruction

```text
Use `media.motion.track` when movement is the evidence.
Return JSON.
Reference motion artifacts before drawing trails or making movement claims.
```

## Validation

| Check | Expected result |
| --- | --- |
| Motion data | Tracked vectors or points are returned. |
| Source | The result references source video or frames. |
| Claim support | Motion claims cite the returned artifact. |

# media.transcribe

Create transcript artifacts from media audio or video.

## When to use it

Use this when captions, edit notes, search, spoken-word references, or voiceover timing are needed.

## Mental model

Transcription converts audio into structured text artifacts. Model selection and large downloads should stay explicit.

## Inputs

| Field | Meaning |
| --- | --- |
| `input` | Audio or video source path. |
| `outDir` | Directory for transcript artifacts. |
| `model` | Optional explicit transcription model. |

## Output

Returns transcript metadata, text segments, timestamps, and file references.

## CLI fallback

```bash
bun run --cwd packages/os media -- audio transcribe --input artifacts/media/source.mp4 --out artifacts/media/transcript --json
```

## Good agent instruction

```text
Use `media.transcribe` for captions or spoken-word references.
Return JSON.
Verify transcript artifacts exist before citing transcript text.
```

## Validation

| Check | Expected result |
| --- | --- |
| Transcript files | Output paths exist. |
| Segments | Transcript segments include timestamps when available. |
| Source | The transcript references the source media. |

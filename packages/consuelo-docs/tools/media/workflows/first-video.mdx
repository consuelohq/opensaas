# First Video

This workflow proves the media render loop with one source file, one timeline, one render, one QA pass, and one export package.

Use a short local clip first. A controlled source keeps the first pass focused on the pipeline instead of source-capture edge cases.

## Workflow

<Steps>
  <Step title="Ingest or stage the source">
    Use `media.ingest` for a URL or local file. For a local smoke test, keep the source file in a known folder and preserve the path.
  </Step>
  <Step title="Probe the source">
    Run `media.probe` to read duration, dimensions, streams, and codec facts.
  </Step>
  <Step title="Extract review frames">
    Run `media.frames.extract` around the moments you want to use.
  </Step>
  <Step title="Write and validate a timeline">
    Create `media.timeline.v1`, then run `media.timeline.validate`.
  </Step>
  <Step title="Compose, QA, and export">
    Run `media.compose`, `media.qa`, and `media.export` in order.
  </Step>
</Steps>

## Minimal chain

```text
media.ingest
  -> media.probe
  -> media.frames.extract
  -> media.timeline.validate
  -> media.compose
  -> media.qa
  -> media.export
```

## First-video constraints

Use a first video that is easy to inspect:

- 10–60 seconds long
- visible subject or simple scene
- stable enough to extract useful frames
- small enough to render quickly
- available as a local file or permitted test URL

## Agent instruction

```text
Create a first media pipeline smoke using the existing media tools.
Use one source clip.
Run ingest or use the provided local source path.
Run probe, frames extract, timeline validate, compose, QA, and export.
Return every JSON result.
Verify each written file exists.
Fix only bugs that block this pipeline.
```

## Validation

The workflow is complete when these files exist:

| Artifact | Created by |
| --- | --- |
| Source asset or ingest manifest | `media.ingest` |
| Probe metadata | `media.probe` |
| Frame manifest and frame images | `media.frames.extract` |
| Valid timeline | `media.timeline.validate` |
| Rendered video | `media.compose` |
| QA result | `media.qa` |
| Export package | `media.export` |

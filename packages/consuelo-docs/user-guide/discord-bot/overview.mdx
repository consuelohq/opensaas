> Manage calls and queues directly from Discord with the Consuelo bot.

The Consuelo Discord bot brings your CRM calling features into Discord, allowing sales reps to make calls, manage queues, and collaborate with their team without leaving their chat environment.

## Key Features

| Feature                     | Description                                                  |
| --------------------------- | ------------------------------------------------------------ |
| **Power & Preview Dialing** | Queue contacts for power dialing or preview individual leads |
| **Warm & Cold Transfers**   | Transfer calls to managers with context cards                |
| **Whisper Mode**            | Add managers to listen in silently without customer hearing  |
| **Post-Call Disposition**   | Log call outcomes with interactive buttons                   |
| **Channel Notifications**   | Post call events to team channels                            |
| **Real-time Analytics**     | View queue progress and performance stats                    |

## Quick Start

1. **[Invite the bot](/user-guide/discord-bot/getting-started#step-1-invite-the-bot)** to your Discord server
2. **[Link your account](/user-guide/discord-bot/getting-started#step-2-link-your-account)** with `/consuelo login`
3. **[Start calling](/user-guide/discord-bot/capabilities/queue-management)** with `/consuelo queue start`

## Available Commands

| Command                             | Description                           |
| ----------------------------------- | ------------------------------------- |
| `/consuelo login`                   | Link your Discord account to Consuelo |
| `/consuelo logout`                  | Unlink your account                   |
| `/consuelo ping`                    | Check bot status and uptime           |
| `/consuelo me`                      | View your call stats                  |
| `/consuelo status`                  | Check system and queue status         |
| `/consuelo contacts search <query>` | Search CRM contacts                   |
| `/consuelo history`                 | View recent calls                     |
| `/consuelo queue start [category]`  | Start a call queue                    |
| `/consuelo queue pause`             | Pause active queue                    |
| `/consuelo queue resume`            | Resume paused queue                   |
| `/consuelo queue stop`              | Stop queue and show summary           |
| `/consuelo queue status`            | View queue progress                   |
| `/consuelo transfer @user`          | Warm transfer to a manager            |
| `/consuelo whisper @user`           | Add listener in whisper mode          |
| `/consuelo config channel #channel` | Set notification channel              |

<Note>
  The Discord bot requires a Consuelo account with an active workspace. All
  calls are placed through the Consuelo dialer infrastructure.
</Note>

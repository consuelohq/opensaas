> Start, pause, and monitor call queues without leaving Discord.

The Discord bot provides full queue management capabilities, allowing reps to run through call lists efficiently using either power or preview dialing modes.

## Dialing Modes

### Power Dialing

Automatically dials the next contact as soon as you complete the current call.

**How it works:**

1. Bot dials your phone first
2. When you answer, it connects you to the lead
3. After you hang up, the next lead is dialed automatically
4. Post-call card appears for disposition

**Best for:** High-volume outbound calling when you want minimal downtime between calls.

### Preview Dialing

See contact details before deciding to dial.

**How it works:**

1. Bot shows you the next contact's info
2. You review the details
3. Use `/consuelo queue call` when ready
4. Bot dials your phone and connects you

**Best for:** Complex sales where you need context before each call.

## Starting a Queue

```
/consuelo queue start [category]
```

**Examples:**

```
/consuelo queue start                    # Start with first available queue
/consuelo queue start leads              # Start queue for "leads" category
/consuelo queue start prospects preview  # Use preview mode
```

**What happens:**

<Steps>
  <Step title="Select Queue">
    Bot finds an idle queue matching your criteria
  </Step>
  <Step title="Verify Phone">
    Checks your workspace settings for your phone number
  </Step>
  <Step title="Start Calling">Dials your phone and begins the queue</Step>
  <Step title="Track Progress">
    Shows progress as you work through contacts
  </Step>
</Steps>

## Queue Commands

### Pause

```
/consuelo queue pause
```

Temporarily stops auto-dialing. Current call continues, but no new calls are initiated. Useful when you need a break or to handle something urgent.

### Resume

```
/consuelo queue resume
```

Continues the queue from where you left off.

### Stop

```
/consuelo queue stop
```

Ends the queue session and displays a summary:

- Total contacts dialed
- Answer rate
- Average call duration
- Outcome breakdown

### Check Status

```
/consuelo queue status
```

Shows current progress:

```
Queue: Hot Leads (Power Mode)
Progress: 12/45 completed (27%)
Time: 45 minutes elapsed

Current: John Smith вҖ” (555) 123-4567
Last 3 outcomes: вң… вң… вқҢ
```

## Best Practices

- **Set your phone** in workspace settings before starting a queue
- **Use power mode** for high-volume, low-complexity calls
- **Use preview mode** when you need to research each contact
- **Pause between calls** if you need to take notes or update CRM
- **Stop the queue** properly to get your analytics summary

<Note>
  The bot continues running even if you close Discord. Re-open Discord to see
  updates and use commands.
</Note>

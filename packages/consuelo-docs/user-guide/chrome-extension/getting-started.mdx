Get up and running with the Consuelo Chrome Extension in just a few minutes.

## Prerequisites

Before installing the extension, ensure you have:

- **Chrome 116+** — The extension requires Manifest V3 support
- **Consuelo account** — Active subscription at consuelohq.com
- **Provisioned phone number** — At least one Twilio number in your workspace
- **Microphone access** — Required for making calls

<Note>
  Sign in to [consuelohq.com](https://consuelohq.com) first. The extension syncs your session automatically — no separate login required.
</Note>

## Installation

### From Chrome Web Store

1. Visit the [Consuelo Dialer](https://chromewebstore.google.com/detail/consuelo-dialer) page on the Chrome Web Store
2. Click **Add to Chrome**
3. Review the permissions and click **Add extension**
4. The Consuelo icon appears in your browser toolbar

### Verify Installation

Click the Consuelo icon in your toolbar. If you're signed into consuelohq.com, the extension opens directly to the dialer. If not, you'll see a sign-in prompt.

## Sign In

The extension uses session synchronization with consuelohq.com:

<Tabs>
  <Tab title="Already Signed In">
    If you're logged into consuelohq.com in Chrome, the extension automatically authenticates. Your session syncs across tabs — just click the Consuelo icon to start using it.
  </Tab>
  <Tab title="Not Signed In">
    1. Click the Consuelo icon in your toolbar
    2. Click **Sign In with Consuelo**
    3. Complete authentication at consuelohq.com
    4. Return to the extension — you're now authenticated
  </Tab>
</Tabs>

## Using the Side Panel

The side panel is the recommended way to use the extension — it stays open as you browse.

### Open the Side Panel

<Tabs>
  <Tab title="From Toolbar">
    1. Click the Consuelo icon in your toolbar
    2. Click **Open Side Panel** at the top of the popup
    3. The side panel opens on the right side of Chrome
  </Tab>
  <Tab title="From Chrome Menu">
    1. Click the side panel icon in Chrome's toolbar (right side)
    2. Select **Consuelo** from the dropdown
    3. The dialer appears in the side panel
  </Tab>
</Tabs>

### Side Panel Features

The side panel has four tabs:

| Tab | Purpose |
| --- | ------- |
| **Dialer** | Make calls, view active call UI, access coaching panel |
| **Contacts** | Search CRM contacts, view details, quick-dial |
| **Queue** | Select and run dialing queues, view progress |
| **History** | View recent calls with duration and outcomes |

The side panel stays open across page navigation and tab switches, making it ideal for long calling sessions.

## Using the Popup

The popup provides quick access for single calls:

1. Click the Consuelo icon in your toolbar
2. The popup opens (400×600 pixels)
3. Use the dial pad or search for a contact
4. Click to call

<Warning>
  The popup closes when you click outside it. For calls in progress, the active call view remains accessible in the side panel.
</Warning>

## Making Your First Call

### Quick Dial

1. Open the side panel or popup
2. Go to the **Dialer** tab
3. Enter a phone number using the dial pad or paste from clipboard
4. Select your **calling number** (which number to call from)
5. Click the green **Call** button

### Call from Contact

1. Go to the **Contacts** tab
2. Search for a contact by name, phone, or company
3. Click the phone icon next to a contact
4. The dialer initiates the call

### During the Call

When a call connects:

- **Call timer** shows duration in the header
- **Mute button** toggles your microphone
- **Hang up button** ends the call
- **Coaching panel** appears with real-time AI tips (if enabled)

## Selecting a Calling Number

If your workspace has multiple provisioned phone numbers, select which one to call from:

1. Look at the footer in the dialer
2. It shows your current calling number
3. Click to open the number selector
4. Choose a different number

This affects your caller ID — the person you call sees this number.

## Google Sheets Import

Import contacts directly from a Google Sheet:

### Step 1: Open a Google Sheet

Navigate to a Google Sheet containing contact data (name, phone, email, etc.)

### Step 2: Initiate Import

1. Click the Consuelo icon
2. Go to the **Contacts** tab
3. Click **Import from Sheets** (or the import icon)
4. Grant Google Sheets permission if prompted

### Step 3: Map Columns

The import flow shows:

1. **Preview** — First few rows of your data
2. **Column Mapping** — Match spreadsheet columns to Consuelo fields:
   - **Name** (required)
   - **Phone** (required)
   - **Email** (optional)
   - **Company** (optional)
   - **Notes** (optional)
   - **Tags** (optional)

### Step 4: Import

Click **Import** to add contacts to your CRM. Results show:
- Successfully imported count
- Any duplicates skipped
- Errors (invalid phone numbers, etc.)

<Note>
  Google Sheets import requires the optional "Read your data on Google Sheets" permission. You grant this the first time you use the feature — the extension requests it at that moment.
</Note>

## Troubleshooting

### Extension Shows "Sign In" But I'm Logged Into consuelohq.com

Session sync can take a moment. Try:
1. Refresh the consuelohq.com tab
2. Close and reopen the extension popup
3. Sign out of consuelohq.com and sign back in

### Calls Won't Connect

Check:
1. **Microphone permission** — Chrome must allow microphone access
2. **Calling number** — Ensure a valid number is selected
3. **Network** — Twilio Voice requires a stable internet connection

### Google Sheets Import Not Working

Ensure:
1. You're on a valid Google Sheets URL (docs.google.com/spreadsheets/...)
2. You've granted the Sheets permission when prompted
3. The spreadsheet has data with header rows

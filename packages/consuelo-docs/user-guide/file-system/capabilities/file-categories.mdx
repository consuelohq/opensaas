## About File Categories

Every file in Consuelo has a **category** that determines which prebuilt view it appears in. Categories are set automatically based on the file type when you upload, but you can change them manually.

## Standard Categories

### Text Document
PDFs, Word documents, and plain text files. This is the default for most document uploads.

**Common uses:** Contracts, proposals, meeting notes, SOWs, NDAs, reports.

### Spreadsheet
Excel files, CSVs, and other tabular data.

**Common uses:** Financial forecasts, contact lists, inventory, data exports.

### Presentation
PowerPoint and slide files.

**Common uses:** Pitch decks, quarterly reviews, training materials, company overviews.

### Image
PNG, JPG, and other image files.

**Common uses:** Product photos, screenshots, diagrams, wireframes, logos.

### Audio
Audio files of any format.

**Common uses:** Podcast episodes, voice memos, audio notes.

### Video
Video files of any format.

**Common uses:** Demo recordings, training videos, product walkthroughs.

### Archive
ZIP files and compressed archives.

**Common uses:** Project bundles, backup data, source code packages.

## Sales-Specific Categories

### Recording
Call recordings and meeting recordings. These appear in the **Recordings** prebuilt view.

**Common uses:** Discovery calls, demo recordings, coaching review sessions, customer interviews.

<Note>
Recording transcription is coming soon. Recordings will be automatically transcribed and indexed for semantic search.
</Note>

### Script
Sales scripts, email templates, and call frameworks. These appear in the **Scripts** prebuilt view.

**Common uses:** Cold call opening scripts, objection handling guides, follow-up email templates, voicemail scripts.

### Agent File
Configuration files for AI agents. These appear in the **Agent Files** prebuilt view.

**Common uses:** AI coaching steering files, qualification checklists, discovery question frameworks, team methodology documents (MEDDIC, BANT, etc.), skill definitions.

**Why this matters:** When you connect an external AI agent (like ChatGPT, Claude, or a custom agent) to Consuelo via the GraphQL API, the agent reads these files to understand your team's sales methodology. Updating an agent file immediately changes how the agent behaves — no code changes needed.

### Folder
Used when uploading an entire folder. The folder contents are stored as a single file entry.

**Common uses:** Project directories, multi-file uploads, document bundles.

## Changing a File's Category

1. Click on a file to open its detail page
2. Find the **File Category** field
3. Click to change the category from the dropdown
4. The file will now appear in the corresponding prebuilt view

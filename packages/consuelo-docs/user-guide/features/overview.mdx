# Features

Consuelo OS gives agents a safe gateway to tools, workflows, files, approvals, context, and human-facing products.

This page is a placeholder for the feature map. The current docs are being reorganized around the product model first; the detailed copy will follow.

## Core feature areas

- Tool Gateway
- Built-in tool packs
- Sites and GTM workflows
- Human and agent workflows
- Files, artifacts, and approvals
- Context and memory
- Open-source extensibility

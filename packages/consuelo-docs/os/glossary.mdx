# Glossary

## OS

Consuelo OS is the operating layer that turns a company into an agent-ready workspace.

## Portal

The safe entry point agents use to understand the workspace and run approved skills.

## Skill

An approved workflow an agent can run through Consuelo OS.

## Script

Executable implementation behind a skill.

## Capability

A configured system the OS can use, such as local storage, cloud storage, Slack approvals, GraphQL/API access, or an external integration.

## Artifact

An app-native output with metadata, provenance, storage, and observability.

## Approval

A human decision required before a high-impact action continues.

## Trace ID

A unique ID attached to each OS call or skill execution so it can be debugged later.

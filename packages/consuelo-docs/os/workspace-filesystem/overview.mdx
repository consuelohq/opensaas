> The workspace filesystem is the file surface for OS skills and agents.

## Decision

Consuelo OS needs a mounted Linux workspace filesystem for skills and agents.

Targets:

- local filesystem for development
- S3 Files-mounted workspace for production
- current S3 object storage as compatibility/fallback

## Canonical layout

```text
/workspace
  /steering
  /business
  /files
  /artifacts
  /skill-runs
  /campaigns
  /landing-pages
  /ads
  /calls
  /transcripts
  /reports
```

## Rule

Skills should use workspace paths and filesystem/artifact APIs. Avoid raw object keys and random machine-local paths in skill logic.

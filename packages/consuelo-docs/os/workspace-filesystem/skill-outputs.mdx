> Skill outputs are structured results returned by call.

## Output shape

Skills should return structured data with status, summary, data used, recommendations, artifacts, proposed writes, approval requirements, and errors.

## Durable output

When a skill creates a file, it should return an artifact reference. When a skill proposes a record update, it should return proposed write data.

When a skill cannot complete, it should return a structured status such as `missing_env`, `not_configured`, or `query_failed`.

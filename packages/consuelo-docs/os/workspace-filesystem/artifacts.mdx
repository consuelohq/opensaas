> Artifacts are durable runbook outputs.

## Definition

An artifact is a durable file-like output created by a runbook or agent.

Examples include daily revenue briefs, campaign briefs, landing page drafts, ad variants, call summaries, coaching reports, CSV outputs, and analysis results.

## Metadata

Artifacts should include path, type, content type, workspace id when available, run id when available, creator, and source runbook.

## Rule

Runbook results should return artifact references, not only raw text.

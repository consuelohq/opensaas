> The high-level architecture for Consuelo OS.

## Architecture shape

```text
Agent -> OS portal entrypoints -> get_steering / get_dev_steering / call -> skill manifest -> Bun runtime -> permission gate -> GraphQL facades / filesystem / sandbox / integrations -> artifacts / records / reports / logs
```

## Principle

Small OS portal entrypoints, powerful classified OS runtime behind them.

## Layers

- OS portal: `get_steering`, `get_dev_steering`, and `call`.
- Runtime: manifest lookup, validation, permissions, Bun execution, artifacts, and structured results.
- Data layer: GraphQL/backend API facades for workspace data.
- Filesystem: workspace paths for files, artifacts, reports, run outputs, landing pages, ads, transcripts, and business context.
- Sandbox: isolated execution for scripts, file workflows, and analysis.
- Operator: provisioning, debugging, logs, health checks, and skill installation.

> The first operating flow for connecting an agent to Consuelo OS.

## First flow

1. Connect an agent to the Consuelo OS OS portal.
2. Run `get_steering` for revenue/business work or `get_dev_steering` for build/design/operator work.
3. Review available skills from the manifest included in steering.
4. Run `call` with a documented skill name.
5. Review the result, artifacts, proposed writes, and approval requirements.

## First smoke command

```json
{"name":"daily-revenue-brief","input":{}}
```

Expected behavior: manifest resolution, permission metadata, Bun execution, optional GraphQL connectivity check, structured result, and no secret output.

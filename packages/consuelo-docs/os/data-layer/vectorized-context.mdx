> Vector search handles fuzzy business context.

## Use vectorized context for

- call transcripts
- notes
- files
- scripts
- landing pages
- ad copy
- campaign briefs
- objection libraries
- reports
- skill docs
- business steering

## Rule

Vector search complements structured queries. It does not replace the data model.

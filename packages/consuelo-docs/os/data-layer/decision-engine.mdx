> The decision engine helps agents decide what to do next.

## Purpose

The decision engine turns business context into next actions.

## Inputs

- structured records
- semantic context
- runbook rules
- permissions
- recent outcomes
- call analytics
- campaign performance
- user goals

## Output examples

- who to call next
- what follow-up is missing
- which campaign needs review
- which landing page needs revision
- what manager should inspect

## Rule

Retrieval is a prior. Evidence and outcomes update the decision.

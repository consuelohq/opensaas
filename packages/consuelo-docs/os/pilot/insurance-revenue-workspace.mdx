> The first pilot shape for an insurance/revenue team using Consuelo OS.

## Target customer

Insurance or revenue team running calls, leads, landing pages, Meta/Google ads, follow-ups, and manager reporting.

## What gets set up

- workspace
- users and roles
- contacts/lists
- call activity
- call analytics
- files/artifacts
- business steering
- skill manifest
- agent connection
- required integrations

## Initial skills

- daily revenue brief
- lead prioritizer
- post-call analysis
- sales coaching
- follow-up generator
- landing page builder
- Meta Ads review
- Google Ads review
- campaign brief
- weekly manager report

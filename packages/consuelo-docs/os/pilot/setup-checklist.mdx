> Setup checklist for the first Consuelo OS pilot workspace.

## Workspace

- create or select workspace
- confirm owner
- add users
- assign roles
- confirm agent identities if needed

## Data

- import or connect contacts
- confirm lists
- confirm call records
- confirm tasks/notes
- confirm campaign/landing-page context

## Runtime

- configure OS portal
- configure `get_steering`
- configure `get_dev_steering`
- configure `call`
- install skill manifest
- run `daily-revenue-brief` smoke

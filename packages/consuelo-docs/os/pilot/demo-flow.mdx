> The first demo flow for proving Consuelo OS to a customer.

## Flow

1. Connect ChatGPT, Claude, or another agent.
2. Run `get_steering`.
3. Ask for the daily revenue brief.
4. Run `call("daily-revenue-brief")`.
5. Ask who should be called first and why.
6. Run lead prioritization when available.
7. Draft follow-ups or tasks.
8. Generate a campaign or landing page brief.
9. Review artifacts.
10. Approve or reject proposed writes.

## What this proves

The agent can orient itself, use real workspace context, run OS runtime logic, create artifacts, and expose permissions.

# Consuelo OS

Consuelo OS turns your company into an agent-ready workspace.

It is the operating layer that gives agents safe access to context, tools, memory, permissions, and approved workflows.

Agents do not get direct access to your company systems. They work through the Consuelo OS portal: one entry point to understand the workspace and one entry point to run approved skills.

## What OS provides

- Workspace context agents can understand
- Skills agents can run safely
- Scripts that implement those skills
- Files and app-native artifacts
- Approval guardrails for high-impact actions
- Local and cloud runtime options
- Execution logs and trace IDs

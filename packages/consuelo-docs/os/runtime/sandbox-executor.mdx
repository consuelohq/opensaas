> The sandbox executor is the cloud computer for isolated OS work.

## Purpose

The sandbox runs file, script, analysis, and artifact workflows safely.

## Target behavior

- customer-scoped execution
- user-scoped context when available
- workspace context injection
- file/artifact output
- logs
- cleanup
- no Mac mini dependency in production

## Use cases

Analyze campaign files, generate reports, process transcripts, create landing page drafts, transform CSVs, and run scripted checks.

> Operator scripts manage OS installs before a polished dashboard exists.

## Purpose

Operator scripts give the internal Consuelo team a way to provision, inspect, debug, and maintain OS workspaces.

## Initial operator needs

- provision workspace
- install runbooks
- configure/check env
- inspect logs
- inspect artifacts
- run health check
- test runbook
- disable runbook
- rotate secret

## Rule

Operator scripts are not automatically MCP tools. Expose operator capabilities through `get_dev_steering` or admin-scoped runbooks when appropriate.

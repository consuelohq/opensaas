> Operator scripts manage OS installs before a polished dashboard exists.

## Purpose

Operator scripts give the internal Consuelo team a way to provision, inspect, debug, and maintain OS workspaces.

## Initial operator needs

- provision workspace
- install skills
- configure/check env
- inspect logs
- inspect artifacts
- run health check
- test skill
- disable skill
- rotate secret

## Rule

Operator scripts are not automatically OS portal entrypoints. Expose operator capabilities through `get_dev_steering` or admin-scoped skills when appropriate.

> Bun is the primary runbook runtime for Consuelo OS.

## Decision

Bun is the runbook runtime.

## Responsibilities

- manifest lookup
- input validation
- permission checks
- facade calls
- filesystem/artifact writes
- sandbox orchestration
- structured result output
- logs

## Rule

MCP stays thin. Bun handles runbook logic.

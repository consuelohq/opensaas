> Bun is the primary skill runtime for Consuelo OS.

## Decision

Bun is the skill runtime.

## Responsibilities

- manifest lookup
- input validation
- permission checks
- facade calls
- filesystem/artifact writes
- sandbox orchestration
- structured result output
- logs

## Rule

The OS portal stays thin. Bun handles skill logic.

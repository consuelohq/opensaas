# OS portal

The OS portal is how agents work with Consuelo OS.

Agents use the portal to understand the workspace and run approved skills. They do not receive direct access to raw company systems.

## Two entry points

The portal has one entry point for context and one entry point for action.

Context tells the agent what workspace it is operating in, which skills are available, and which capabilities are healthy.

Action runs an approved skill and returns a structured result.

# Skills

Skills are approved workflows that agents can run through Consuelo OS.

A skill can read approved context, use tools, execute scripts, create artifacts, ask for approval, and return structured results.

## Skills are not raw scripts

A script is executable code. A skill is the safe workflow around that code: what it does, what context it needs, what it can create, what permissions apply, and how results are returned.

## Skill docs should explain

- What the skill does
- What context it reads
- What it can create
- Which capabilities it needs
- Which approvals can be required
- What result shape it returns

# Context and memory

Consuelo OS gives agents the context they need to do useful work without exposing every system directly.

Structured records are the source of truth. Memory and vector context help agents find relevant supporting context, but they should not replace structured records when exactness matters.

## Structured first

People, companies, tasks, notes, calls, opportunities, approvals, artifact records, and execution logs should stay structured.

## Memory second

Memory is useful for summaries, preferences, prior decisions, and reusable context. It should not store secrets or become the only source of truth for business records.

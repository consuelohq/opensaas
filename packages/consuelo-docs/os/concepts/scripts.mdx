# Scripts

Scripts are the executable implementation behind skills.

Most users should think in skills. Builders use scripts to implement and test those skills.

## Script responsibilities

A script should do focused work, return structured output, avoid leaking secrets, and let the OS record execution metadata.

Scripts should not bypass the OS artifact, approval, or observability layers for important work.

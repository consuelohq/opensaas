> Skills are documented business capabilities executed through call.

## Definition

A skill is a named capability behind `call`. Skills are how Consuelo OS turns business context into repeatable work.

## Skill list

- `daily-revenue-brief`
- `lead-prioritizer`
- `post-call-analysis`
- `sales-coaching`
- `follow-up-generator`
- `landing-page-builder`
- `meta-ads-review`
- `google-ads-review`
- `campaign-brief`
- `weekly-manager-report`

## Required doc fields

Each skill page should include purpose, status, permission level, inputs, outputs, artifacts, required integrations, failure modes, approval behavior, and implementation notes.

## Status labels

Use `scaffolded`, `partial`, `ready`, or `blocked` while V1 is being built.

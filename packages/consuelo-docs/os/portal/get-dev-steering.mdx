> Build, design, deployment, and operator steering for technical agents.

## Purpose

`get_dev_steering` gives build/design/operator agents context for landing pages, design assets, authentication, deployment, debugging, and internal OS operations.

## Should include

- project or repo structure where applicable
- Consuelo Design guidance
- landing page workflow notes
- Supabase/auth notes when configured
- deployment/runtime notes
- environment capability matrix
- developer/operator skills
- package/operator script notes
- safe build/deploy rules
- docs links

## Implementation notes

This surface exists because some customer workflows are build/design workflows. Revenue agents use business steering; builder agents use dev steering.

> The manifest is the skill registry for Consuelo OS.

## Purpose

The skill manifest is the primary customer-facing capability registry.

## Current manifest paths

```text
packages/os/tooling/skill-manifest.json
packages/os/tooling/tool-manifest.json
```

`skill-manifest.json` is the OS skill registry used by `call`. `tool-manifest.json` is the restored workspace/operator tool manifest used by `get_dev_steering` and future operator repurposing.

## Required fields

Each skill entry should include name, title, description, category, permission, approval flags, side-effect flags, required env, required integrations, schema placeholders, and implementation pointer.

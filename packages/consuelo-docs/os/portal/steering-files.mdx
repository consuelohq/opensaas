> The durable files that make agents useful inside Consuelo OS.

## Purpose

Steering files are durable context. They keep product, business, runtime, and permission decisions from getting lost across agent sessions.

## Target files

- `STEERING.md`
- `dev-steering.md`
- `business-context.md`
- `data-model.md`
- `permissions.md`
- `skills.md`
- `integrations.md`
- `docs/*.md`

## Rule

Write steering as reusable operating doctrine. Future agents should understand the rule without knowing the chat that created it.
